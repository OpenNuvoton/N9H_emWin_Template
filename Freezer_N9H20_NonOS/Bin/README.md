# Description for Bin Materials

### Binary File

```c

N9H20K5_NANDLoader_192MHz.bin NAND Loader for N9H20@192MHz 32MB RAM
N9H20K5_NVT_NAND_FW043TFT.bin Second loader with display driver for 480x272 LCD
```
